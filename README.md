# 💼 Worker & Task Management Web App

A simple, fast, and lightweight **web-based management system** to handle:
- 👷 Workers data (salary, payments, remaining balance)
- 📝 Personal To-Do list
- 📊 Full financial summary
- 📄 Export data as PDF
- 🖼️ Export as image
- 📤 Share via WhatsApp
- 🌙 Dark mode support

Built using **HTML, CSS, and Vanilla JavaScript** with localStorage for offline persistence.

---

## 🚀 Features

### 👷 Worker Management
- Add workers with:
  - Name
  - Job (including custom job "Other")
  - Paid amount
  - Remaining balance
- Edit & delete workers
- Live search filtering
- Automatic financial summary:
  - Total paid
  - Total remaining
  - Grand total

---

### 📝 To-Do List System
- Add personal tasks
- Mark tasks as completed ✔️
- Delete tasks
- Live progress tracking (% completion)
- Fully stored in localStorage (no internet required)

---

### 📤 Export & Sharing
- 📄 Export full worker data as **PDF**
- 🖼️ Export full account as **Image**
- 📤 Share summary via **WhatsApp**
- Uses `html2canvas` + `jsPDF`

---

### 🌙 UI Features
- Dark / Light mode toggle
- Responsive simple UI
- Clean RTL (Arabic support)
- Fast and lightweight (no frameworks)

---

## 💾 Data Storage

All data is stored locally in the browser using:

- `localStorage.getItem("workers")`
- `localStorage.getItem("todos")`

No server or database needed.
